# ProjectFunder

Crowdfunding-Webanwendung: Benutzer können Projekte erstellen, 
mit Spenden unterstützen und kommentieren. Datenhaltung vollständig 
in IBM DB2 — ohne ORM, mit handgeschriebenem SQL.

**Umfang:** ER-Modellierung, Datenbankschema (CREATE TABLE, Trigger, 
Integritätsbedingungen), SQL-Anfragen und Implementierung der 
Weboberfläche.

**Schwerpunkte:** Transaktionen, Prepared Statements zur Verhinderung 
von SQL-Injection, sauberes Schließen von DB-Ressourcen.

**Kontext:** Praktikum zur Vorlesung Datenbanken, Universität 
Duisburg-Essen, Wintersemester 2019/2020.

**Stack:** Java (oder Python), IBM DB2, SQL, HTML

*Archiviertes Studienprojekt — nicht aktiv weiterentwickelt.*
